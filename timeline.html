    • We'll talk about how Fullstack will be operating - what we'll be doing throughout each month and at the end. We'll also be discussing the topic for the month of November which will be 2D graphics with the HTML5 canvas element. In future months we will be voting for topics to go through, but we think HTML5 graphics is a good topic to start off with. We will be using a simple JavaScript graphics libarary called p5.js which is quite similar to the popular Processing.js library you may have heard of. Although p5js is relatively easy to learn, it can be used to make some pretty cool stuff with the audio and physics libraries that are built in (not to mention the literally-one-line-access to gyroscopic sensors on smart phones).
